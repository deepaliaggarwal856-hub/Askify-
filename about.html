About This Project

This is a Quora Clone, a Question & Answer platform built using HTML, CSS, and JavaScript. Users can post questions, provide answers, upvote/downvote answers, and search for topics. This project is developed as a part of my Semester 1 Capstone project.

The aim of this project is to create a clean, responsive, and user-friendly interface for a Q&A platform, focusing on frontend development. This About Page gives users an overview of the website and its purpose.
